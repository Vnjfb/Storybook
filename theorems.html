Electrical Theorems

Learn the key electrical theorems with their explanations and formulas.

Thevenin Theorem

Thevenin’s Theorem states that any linear electrical network with voltage and current sources and resistances can be simplified to an equivalent circuit with a single voltage source (Vth) in series with a resistance (Rth).

Formula:

  • Vth = Thevenin equivalent voltage
  • Rth = Thevenin equivalent resistance

Norton Theorem

Norton’s Theorem is a dual of Thevenin’s Theorem. It states that any linear electrical network can be simplified to an equivalent current source (In) in parallel with a resistance (Rn).

Formula:

  • In = Norton equivalent current
  • Rn = Norton equivalent resistance

Superposition Theorem

Superposition Theorem states that in a linear circuit with multiple sources, the current (or voltage) in any component can be found by considering one source at a time and summing the individual effects.

Formula:

  • For each source: I = (V/R)
  • Sum all the individual currents or voltages to find the total effect.

Reciprocity Theorem

Reciprocity Theorem states that in a linear network, the current that would flow in one branch due to a voltage source in another branch will be the same as the current that would flow in the second branch if the same voltage were applied to the first branch.

Formula:

  • Iab = Iba (current in one branch is equal to the current in the other branch)

Maximum Power Transfer Theorem

The Maximum Power Transfer Theorem states that the maximum power will be delivered to the load when the load resistance (Rl) is equal to the source resistance (Rs) of the circuit.

Formula:

  • Rl = Rs (Load resistance equals the source resistance for maximum power transfer)
  • Maximum Power: Pmax = (V^2) / (4 * Rs)
